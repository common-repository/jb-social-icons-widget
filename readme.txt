== Installation ==


1. Upload the entire `jb-social-icons-widget` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Appearence > Widget then add JB Social Icons widget into any of your sidebar and add social media link into widgets fields. 
4. That's it
